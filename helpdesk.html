Akun & Profil
Bagaimana cara mendaftar di GRCC Academy?
+
Untuk mendaftar, klik tombol "Daftar Sekarang" di halaman utama atau halaman Login. Pilih role "Peserta", isi data diri lengkap (nama, email, institusi, nomor HP), kemudian verifikasi email Anda. Pendaftaran selesai dan akun Anda langsung aktif.
Apakah saya bisa mengubah profil dan foto akun?
+
Ya, Anda dapat mengubah nama, foto profil, institusi, dan data kontak di halaman Pengaturan Profil yang dapat diakses dari sidebar Dashboard. Perubahan email memerlukan verifikasi ulang.
Saya lupa password, bagaimana cara mereset?
+
Klik "Lupa Password?" di halaman Login, masukkan alamat email terdaftar, dan kami akan mengirimkan tautan reset password. Tautan berlaku selama 24 jam. Jika email tidak diterima, periksa folder spam atau hubungi admin.
Satu akun bisa digunakan di berapa perangkat?
+
Satu akun GRCC Academy dapat digunakan di maksimal 3 perangkat secara bersamaan (misalnya laptop, tablet, dan smartphone). Jika login dari perangkat ke-4, sesi perangkat terlama akan otomatis logout.
Program & Kelas
Bagaimana cara mendaftar ke sebuah program pelatihan?
+
Buka halaman Katalog Program, pilih program yang diminati, klik "Detail Program" untuk melihat kurikulum lengkap, lalu klik tombol "Daftar Sekarang". Untuk program berbayar, Anda akan diarahkan ke halaman pembayaran. Setelah konfirmasi, program langsung tersedia di Dashboard Anda.
Apakah ada batas waktu untuk menyelesaikan program?
+
Ya, setiap program memiliki masa aktif yang tertera di halaman detail. Umumnya program ERM Fundamental memberikan akses 6 bulan sejak tanggal pendaftaran. Modul dan materi dapat diakses ulang selama masa aktif.
Apakah materi modul bisa dibaca secara offline?
+
Materi berbasis teks dapat dibaca langsung di platform tanpa unduh. Untuk penggunaan offline, Anda dapat mengunduh file PDF modul dari halaman Materi Unduhan. Namun quiz, ujian, dan progres hanya dapat direkam saat online.
Apakah tersedia sesi live/webinar dengan instruktur?
+
Beberapa program premium menyertakan sesi live Q&A dengan instruktur yang dijadwalkan via Zoom. Jadwal sesi live akan muncul di bagian "Jadwal" pada Dashboard Anda minimal 3 hari sebelum pelaksanaan. Rekaman tersedia 48 jam setelah sesi selesai.
Bisakah saya mengambil lebih dari satu program sekaligus?
+
Ya, tidak ada batasan jumlah program yang dapat diambil bersamaan. Semua program yang sedang berjalan akan tampil di Dashboard Anda dengan progres masing-masing. Kami menyarankan untuk fokus pada maksimal 2-3 program agar hasil optimal.
Ujian & Sertifikat
Berapa kali saya bisa mengulang ujian?
+
Pre-test dapat dikerjakan 1 kali. Post-test dapat dikerjakan maksimal 3 kali. Final Exam dapat dikerjakan maksimal 2 kali — jika tidak lulus di percobaan pertama, ada masa tunggu 7 hari sebelum bisa mencoba kembali. Nilai tertinggi yang akan dicatat.
Berapa nilai minimum untuk lulus dan mendapat sertifikat?
+
Nilai minimum kelulusan adalah 70/100 untuk Final Exam. Anda juga harus menyelesaikan minimal 80% dari seluruh modul dan lulus Post-test dengan nilai minimal 60. Sertifikat otomatis diterbitkan dalam 24 jam setelah semua persyaratan terpenuhi.
Bagaimana cara mengunduh sertifikat digital?
+
Sertifikat dapat diakses dari menu "Sertifikat" di sidebar Dashboard, atau dari halaman Sertifikat langsung. Klik tombol "Unduh PDF" untuk mengunduh dalam format PDF, atau "Cetak" untuk mencetak langsung. Sertifikat dilengkapi QR code untuk verifikasi keaslian.
Apakah sertifikat GRCC Academy diakui secara resmi?
+
Sertifikat GRCC Academy diakreditasi oleh GRCC (Gerakan Riset untuk Rakyat dan Cita-cita) dan diakui oleh berbagai instansi pemerintah, BUMN, dan perusahaan swasta yang menjadi mitra GRCC. Sertifikat berlaku seumur hidup dan dapat diverifikasi online melalui verify.grccacademy.id.
Koneksi internet terputus saat mengerjakan ujian, bagaimana?
+
Sistem kami menyimpan progres ujian setiap 60 detik secara otomatis. Jika koneksi terputus, tutup dan buka kembali browser — ujian akan dilanjutkan dari soal terakhir yang tersimpan. Timer tetap berjalan selama jeda. Jika terjadi kendala teknis, hubungi admin segera dengan screenshot bukti.
Teknis & Platform
Browser apa yang direkomendasikan untuk menggunakan GRCC Academy?
+
GRCC Academy dioptimalkan untuk Chrome (versi 100+), Firefox (versi 100+), Edge (versi 100+), dan Safari (versi 15+). Kami sangat menyarankan penggunaan Chrome atau Edge terbaru untuk pengalaman terbaik, terutama saat mengerjakan ujian dengan timer.
Platform tidak bisa dibuka / loading lama, apa solusinya?
+
Coba langkah berikut: (1) Hard refresh halaman (Ctrl+Shift+R / Cmd+Shift+R), (2) Bersihkan cache browser, (3) Nonaktifkan ekstensi browser sementara, (4) Coba browser lain, (5) Periksa koneksi internet. Jika masalah berlanjut, hubungi tim teknis kami dengan menyertakan nama browser dan pesan error yang muncul.
Apakah platform bisa diakses dari smartphone?
+
Ya, GRCC Academy sepenuhnya responsif dan dapat diakses dari smartphone Android maupun iOS. Untuk pengalaman membaca modul dan mengerjakan ujian yang optimal, kami menyarankan penggunaan tablet atau laptop.
Panduan Langkah demi Langkah
01
Panduan Mendaftar & Mengikuti Kelas
  1. Klik "Daftar Sekarang" di halaman utama
  2. Isi formulir pendaftaran lengkap
  3. Verifikasi email dari inbox Anda
  4. Login dan buka Katalog Program
  5. Pilih program, klik "Daftar Sekarang"
  6. Akses modul dari Dashboard
Lihat Katalog
02
Panduan Mengerjakan Ujian
  1. Selesaikan semua modul terlebih dahulu
  2. Buka menu "Ujian" di sidebar
  3. Pilih jenis ujian (Pre/Post/Final)
  4. Baca instruksi dengan teliti
  5. Klik "Mulai Ujian" — timer dimulai
  6. Jawab semua soal, klik "Selesai"
  7. Lihat hasil dan nilai Anda
Ke Halaman Ujian
03
Panduan Mengunduh Sertifikat
  1. Pastikan Final Exam telah lulus (≥70)
  2. Tunggu 24 jam untuk penerbitan sertifikat
  3. Buka menu "Sertifikat" di sidebar
  4. Klik sertifikat yang ingin diunduh
  5. Klik "Unduh PDF" atau "Cetak"
  6. Verifikasi keaslian via QR code
Lihat Sertifikat
04
Panduan Mengunduh Materi
  1. Pastikan sudah terdaftar di program terkait
  2. Buka menu "Materi Unduhan" di sidebar
  3. Filter berdasarkan jenis file (PDF, PPTX, XLSX)
  4. Klik "Preview" untuk melihat isi file
  5. Klik "Unduh" untuk menyimpan ke perangkat
  6. Gunakan untuk belajar offline
